Each client is defined by id, name, code_name, secret 

*code_name* is used internally to fetch data from the right service

*Example*
{% highlight json %}
{
	id: 1,
	name: "Alcohol Delivery",
	code_name: "dyonisus"
}
{% endhighlight %}
